          本工具能做什么
          
          
            
            
          • 按分类管理 source / inbox / Unity 导出路径
            • 
            
          • ComfyUI 本地 SDXL 生图，或批量调用云 API（即梦、Stability 等）
            • 
            
          • DeepSeek AI 助手：生成与优化提示词、工作流建议
            • 
            
          • 后处理工作台与一键导出到游戏工程
            • 
          
          
        
          
        
          
          
          首次使用请填写 API Key
          
          
          前往「全局设置」Tab 配置以下密钥（仅存于本地 pipeline_config.json）：
          
          
            
            
          • DeepSeek API Key — 启用 AI 助手
            • 
            
          • 云生图密钥 — 按资源所选服务商填写（即梦 / Stability / 腾讯 / 阿里等）
            • 
            
          • ComfyUI URL — 本地生图需服务在线（默认 http://127.0.0.1:8188）
            • 
          
          
        
          
        
          
          
          全局路径需要填写什么
          
          
          同样在「全局设置 → 路径与工作区」中配置：
          
          
            
            
          • 游戏引擎项目根 — Unity 工程根目录，导出资源的目标位置
            • 
            
          • ArtPipeline 根 — 含 source、inbox、workflows 的流水线目录
            • 
            
          • 运行日志目录 — 可选；留空则使用系统默认位置
            •
