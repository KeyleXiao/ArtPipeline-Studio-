# ArtPipeline Studio · 更新摘要

## 近期功能

### 导入外部资源

- 资源列表 **+ 新建资源** → 页签 **导入外部资源**
- 支持多选 PNG / JPG / WebP / GIF
- 列表预览、单独移除、按文件名批量创建
- 自动写入 **source** 与 **inbox**（统一转为 PNG），重名跳过

### 运行日志目录

- **全局设置** 可配置 **运行日志目录**（留空 = 系统默认）
- 日志持久化到 `studio.log`，重启后恢复最近记录
- 默认路径：
  - macOS：`~/Library/Logs/ArtPipeline Studio/`
  - Windows：`%LOCALAPPDATA%\ArtPipeline Studio\Logs\`
- 设置页可 **打开目录** 直接查看日志文件

### 交互与体验

- 生图 **浮动进度条**（可拖拽），支持停止任务
- 资源 **右键菜单**：单个生图 / 生成并导出 / 导出（无需列表启用）
- AI 助手对话 **自动保存**；顶栏精简（生成/导出移至右键）
- 新建分类 **Checkpoint** 下拉与表单风格统一

### 构建

- Windows CI：修复 PyInstaller 成功后 `print` 中文导致的编码错误（`safe_print`）
