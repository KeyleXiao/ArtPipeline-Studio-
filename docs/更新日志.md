# ArtPipeline Studio · 更新摘要

## 近期功能

### 云生图（Stability / 万相 / 混元 / 即梦）

- 支持 **Stability AI**（国外）与 **万相 / 混元 / 即梦**（国内）
- Checkpoint 下拉混排本地与 `cloud:` 云端模型
- 三种模式统一命名：**文生图 / 图生图 / 图像编辑**
- 云任务并发执行 + 浮动进度条多任务状态
- ComfyUI 生成路径 **未改动**

文档：[docs/cloud-generation.md](cloud-generation.md) · 注册表：[tools/cloud/registry.json](../tools/cloud/registry.json)

### 导入外部资源

- 资源列表 **+ 新建资源** → 页签 **导入外部资源**
- 支持多选 PNG / JPG / WebP / GIF
- 列表预览、单独移除、按文件名批量创建
- 自动写入 **source** 与 **inbox**（统一转为 PNG），重名跳过

![导入外部资源](../docs/images/import-external-assets.png)

### 运行日志目录

- **全局设置** 可配置 **运行日志目录**（留空 = 系统默认）
- 日志持久化到 `studio.log`，重启后恢复最近记录
- 默认路径：
  - macOS：`~/Library/Logs/ArtPipeline Studio/`
  - Windows：`%LOCALAPPDATA%\ArtPipeline Studio\Logs\`
- 设置页可 **打开目录** 直接查看日志文件

![运行日志](../docs/images/runtime-logs.png)

### 交互与体验

- 生图 **浮动进度条**（可拖拽），支持停止任务
- 资源 **右键菜单**：单个生图 / 生成并导出 / 导出（无需列表启用）
- AI 助手对话 **自动保存**；顶栏精简（生成/导出移至右键）
- 新建分类 **Checkpoint** 下拉与表单风格统一
